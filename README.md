# CivicSync

Smart Municipal Complaint Management and Analytics Platform

## Problem Statement

Citizens often report civic issues through disconnected channels, making tracking and resolution difficult. Municipal authorities also lack analytical insights for resource allocation and decision making.

## Solution

CivicSync provides a centralized platform where citizens can report civic issues, officers can manage complaints, and administrators can monitor performance through analytics dashboards.

## Features

* JWT Authentication
* Role Based Access Control
* Complaint Management
* Officer Assignment
* Complaint Tracking
* Analytics Dashboard
* Priority Engine
* Resource Recommendation System

## Tech Stack

Backend:

* Java
* Spring Boot
* Spring Security
* JWT
* Hibernate/JPA

Frontend:

* React
* Bootstrap

Database:

* MySQL

Analytics:

* Chart.js
